## Ключевой опыт

### Яндекс — ML Engineer

Работал над внутренними AI-продуктами и интеграциями для нескольких направлений.

Основные кейсы:
- **HR Insight** — end-to-end сервис NL→SQL: парсинг запросов, безопасное выполнение, аналитика на DuckDB/SQLite, интерпретация результатов на естественном языке, базовый прогноз оттока

- **AI-Дизайнер** — генерация и редактирование визуального контента, ресайзинг, in-painting, работа с референсами и шаблонами
- **AI-Пиарщик** — мониторинг Telegram-каналов, кластеризация, дедупликация, суммаризация, алерты и дайджесты
